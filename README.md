# truss-analyzer
Developed a Python-based tool to compute forces in truss members using matrix methods as part of a 4-member team.

Visualized tensile and compressive forces using color-coded graphical output using Matplotlib.

## Features
- Calculates member forces (tension/compression)
- Visualizes truss with color-coded forces
- Accepts joint coordinates and connectivity

## Tools Used
- Python
- NumPy
- Matplotlib

## Contribution
Developed as part of a 4-member team for ES221 (Mechanics of Solids)
